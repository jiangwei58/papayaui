## 异步

```html [template]

<pa-input-number :model-value="num" async-change @change="onChange" />

```
```ts [script]

import { ref } from 'vue'
import { sleep } from '../../utils'

const num = ref(1)

const onChange = async (value: number) => {
  uni.showToast({
    icon: 'loading',
    title: '',
    duration: 2000,
  })
  await sleep(2000)
  num.value = value
  uni.hideToast()
}

```

## InputNumber Props

<!--props start-->

| 参数 | 说明 | 类型 | 默认值 |
| --- | ----- | --- | --- |
| modelValue | 值 | InputNumberValue | - |
| name | 标识符 | string |  '' |

| inputAlign | 内容对齐方式 | 'left'\|'center'\|'right' |  'center' |
